# plug_and_play_modules.py 使用说明

该文件从 `geoseg/models/networks/mamba.py` 中抽取了 GLVMamba 论文结构图中的几个“即插即用”模块，便于在其它工程中快速复用或单独调试。

## 包含的模块
- `SELayer` / `LocalityFeedForward`：对应 Fig.5 和 Fig.4(c) 的局部增强分支，可直接嵌入任意 CNN。
- `SS2D`：Fig.3 的二维 Selective Scanning。依赖 `mamba_ssm`；若未安装，会在测试函数中提示跳过。
- `VSSBlock`：Fig.4(a) 的基础视觉状态空间块。
- `GLSS2D`：Fig.4(c) 的全局-局部融合块，内部含 SS2D + LocalityFeedForward。
- `GLVSSBlock`：Fig.4(b) 的全局-局部 VSS 组合，外部仍是标准残差结构。

> 由于 GLSS2D 需要将通道拆成 “全局/局部” 两份，`dim` 必须为偶数；此外 SS2D 若缺少 CUDA 扩展会退化到 reference 实现，速度较慢。

## 快速测试
1. 确保当前目录在 `glvmamba/`。
2. 运行：
   ```bash
   python plug_and_play_modules.py
   ```
3. 你将看到类似输出，表示各模块能独立前向：
   ```
   SELayer output: (2, 32, 16, 16)
   LocalityFeedForward output: (2, 32, 16, 16)
   SS2D output: (2, 8, 8, 32)   # 若缺少 mamba_ssm 则会提示跳过
   VSSBlock output: (2, 8, 8, 64)
   GLSS2D output: (2, 8, 8, 64)
   GLVSSBlock output: (2, 8, 8, 64)
   ```
